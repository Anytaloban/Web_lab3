# Web_lab3
Виконала Лобань Анна, КА-31.
В цій частині було сформовано файли CSS для розроблених раніше файлів HTML. В роботі використовувались засоби Flexbox для формування структурно складних макетів з декількох елементів різного типу та призначення. За допомогою Flexbox налагоджено прдеставлення каталогів, навігаційної панелі та форми реєстрації. Під час роботи для забезпечення кросбраузерності усі значення вимірів вводилися в одиницях vw, які підлаштовуються формат зображення/контейнера/тексту під відсоткове значення ширини екрану, забезпечуючи таким чином оптимальність подання. Для оформлення форми реєстрації використовуються атрибути HTML5 для відображення помилок під час заповнення текстових полів. У файлах css також реалізоване імпортування файлів зі шрифтами з брендбуку з папки fonts, яка знаходиться в папці css, за допомогою директиви @font-face.
